# Contributing / 기여하기

Thank you for your interest in Liquid QR Coder! Your contributions make this project better. / Liquid QR Coder 프로젝트에 관심을 가져주셔서 감사합니다! 여러분의 기여가 이 프로젝트를 더욱 풍부하게 만듭니다.

---

## How to Contribute / 기여 방법

1.  **Report Issues / 이슈 보고**: If you find a bug or have a feature request, please leave details on the [Issues page]. / 버그를 발견하거나 새로운 기능을 제안하고 싶다면 이슈 페이지를 이용해 주세요.
2.  **Pull Request**:
    *   Fork the repository. / 저장소를 포크합니다.
    *   Create a new branch (`git checkout -b feature/AmazingFeature`). / 새 브랜치를 생성합니다.
    *   Commit your changes (`git commit -m 'Add AmazingFeature'`). / 변경 사항을 커밋합니다.
    *   Push to the branch (`git push origin feature/AmazingFeature`). / 브랜치에 푸시합니다.
    *   Open a Pull Request. / Pull Request를 생성합니다.

## Coding Rules / 코딩 규칙
*   Adhere to C++17 standards. / C++17 표준을 준수합니다.
*   Follow the Liquid Glassmorphism design system. / Liquid Glassmorphism 디자인 시스템을 존중합니다.
*   Write comments in both English and Korean (English first). / 모든 주석은 영어-한국어 병기로 작성합니다 (영어 먼저).
